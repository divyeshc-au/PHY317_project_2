# photonics_project_2

This is a Python script in a venv that uses the macromax module to visualise electric and magnetic fields in rectangular metal waveguides. 

Enter these commands to create a virtual environment and install the necessary packages in it:

$ python3 -m venv local_python_environment 

(local_python_environment) $ pip install -r requirements.txt 

(local_python_environment)/bin/python {filename}.py
